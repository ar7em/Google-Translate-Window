This application displays Google Translate webpage (translate.google.com)
as a separate stay-on-top window.

Application starts as tray icon and can be restored (and hidden) 
by clicking on that icon. Pressing 'X' button at main window will hide
application to tray.

Any link pressed at the translation window will be delegated to your system
browser.

This application is for personal non-commercial use only, all rights
of Google Translate belong to Google.
